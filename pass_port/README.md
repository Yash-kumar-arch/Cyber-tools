# Cybersecurity Tools

A collection of Python cybersecurity tools built from scratch.

## Tools

### 1. Password Strength Checker & Generator
- Checks password strength based on length, uppercase, lowercase, digits and special characters
- Gives improvement suggestions
- Generates strong random passwords

### 2. Port Scanner
- Scans a target IP for open ports
- Supports custom port ranges
- Built using Python socket library

## How to run
python pass_strength_checker.py
python port_scanner.py

## Skills used
Python, Socket Programming, String manipulation
